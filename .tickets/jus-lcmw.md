# Create dade-with-android template

Create a native Android application template using Kotlin and Jetpack Compose. Should follow dade's minimalist philosophy with opinionated library choices.

## Acceptance Criteria

- Template scaffolds correctly with dade new
- Jetpack Compose-first (no XML layouts)
- Kotlin-only codebase
- Modern Android target (API 26+)
- Clean Architecture with MVVM
- Hilt for dependency injection
- AGENTS.md with Kotlin/Compose patterns
- dade.toml properly configured
