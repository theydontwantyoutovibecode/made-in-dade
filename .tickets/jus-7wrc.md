# Create dade-with-ios template

Create a native iOS application template using SwiftUI. Should follow dade's minimalist philosophy with opinionated library choices and deep integration with the dade binary.

## Acceptance Criteria

- Template scaffolds correctly with dade new
- SwiftUI-first architecture (no UIKit)
- Modern iOS target (iOS 17+)
- MVVM or TCA architecture pattern
- Swift Package Manager for dependencies
- AGENTS.md with Swift/SwiftUI patterns
- dade.toml properly configured
